# AI Cyber Threat Intelligence Platform

## Overview
AI-powered cybersecurity platform for threat intelligence analysis, IOC detection, MITRE ATT&CK mapping, and automated security reporting using LLMs and RAG pipelines.

## Features
- Threat Intelligence Analysis
- RAG-based Cyber Knowledge Retrieval
- IOC Extraction
- CVE Analysis
- MITRE ATT&CK Mapping
- AI Security Chat Assistant
- Threat Summarization
- PDF/CSV Report Generation
- Multi-Agent Orchestration
- Real-time Dashboard
